# Server-Side Dev

This repository contains my projects for server-side programming through game development.

## Card Game

### Files
- `Card Game.zip` — source code (project folder)

### How to Run (Local)
1. Download and unzip `Card Game.zip`
2. Open a terminal in the unzipped project root (the folder that contains `package.json`)
3. Install dependencies:
   ```bash
   npm install
4. Start the server:
   ```bash
   node game_server.js
5. Open in browser:
   http://localhost:8000
